Hashids

Parameters

Name Type
salt String
minHashLength Number
alphabet String

Description: The hashids class contains all the methods for encoding. All parameters are optional

Tips: For encoding you can pass a unique salt value so your hashids differ from everyone else's. I use 'this is my salt' as an example.

Hashids.prototype.encode

Parameters

Name Type
ID Number || Numbers || Array[Numbers...]

Description: Encodes the numbers passed in to look like youtube-style hashes

Tips: Make sure your salt is the same for both encoding and decoding

Hashids.prototype.decode

Parameters

Name Type
hash String

Returns

Type
Number

Description: Decodes a hash created by the encode method

Hashids.prototype.encodeHex

Parameters

Name Type
hex String

Returns

Type
String

Description: Encodes a hexcode into a hashed string

Tips: The regular encode function doesn't handle strings. encodeHx doesn't work for all strings, however, only hex values

Parameters
